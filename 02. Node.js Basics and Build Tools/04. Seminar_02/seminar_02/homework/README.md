### Домашняя работа

Функция `solveQuadratic` предназначена для нахождения корней квадратного уравнения вида `ax^2 + b + c = 0`.

### Описание функции solveQuadratic
Функция принимает три аргумента (коэффициенты уравнения) и возвращает массив с результатами:

**Аргументы:**
- a (number) — коэффициент при x^2.
- b (number) — коэффициент при x.
- c (number) — свободный член.

**Результат:**
- "[]" (пустой массив) — если дискриминант отрицательный (корней нет).
- "[x]" (массив с одним числом) — если дискриминант равен нулю (один корень).
- "[x1, x2]" (массив с двумя числами) — если дискриминант положительный (два корня).
